## 配列
### タプル
要素の追加削除ができない
()で要素を囲む

### リスト
要素の追加削除が可能
[]で要素を囲む
+ 変数.append(引数)
  先頭の空の要素から追加する

+ 変数.insert(引数,引数)
  第一引数で追加インデックス値を、第二引数で追加要素を指定
 
+ 変数.remove(引数) 
  引数に該当する最初に該当した要素を削除
  
+ 変数.pop(引数)
  引数のインデックス値の値を取り出す（戻り値として返す）。引数なしだと要素の末尾を取り出す
  
+ 変数.index(引数)
  引数に該当するインデックス値を取得。remoceと同様に最初に該当したインデックス値を取得
